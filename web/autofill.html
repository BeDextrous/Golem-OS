Install the Bookmarklet

Save your profile above first — the bookmarklet code is generated from your data and lives only in your browser.

Arc URL-encodes dragged bookmarklets, breaking them. Use the copy-paste method instead.

1

Copy the bookmarklet code

2

Bookmark any page in Arc

Go to any webpage and press Cmd D

3

Edit that bookmark's URL

Right-click the new favorite → Edit → clear the URL → paste the code → rename it "Autofill" → Enter

4

Done — click it on any job application page

A green toast will confirm how many fields were filled. If you update your profile, repeat steps 1–3 to refresh the bookmarklet.

What gets filled

The bookmarklet scans every visible text, email, tel, and URL input on the page and matches it against your profile using the field's name, id, placeholder, and label text.

Fields already containing a value are never overwritten. A green badge at the top-right will show how many fields were filled (or warn you if none matched).

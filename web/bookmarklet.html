LinkedIn Quick-Add

Save any LinkedIn job to your tracker in one click — without leaving the page.

Install

Arc URL-encodes dragged bookmarklets, breaking them. Use the manual method instead — takes 30 seconds once.

1

Copy the bookmarklet code

2

Bookmark any page in Arc

Go to any webpage and press Cmd D to save it to favorites

3

Edit that bookmark's URL

In Arc's sidebar, right-click the new favorite → Edit → clear the URL field → paste the code you copied → rename it "Save to Golem OS" → press Enter

4

Done — click it on any LinkedIn job page

Arc may ask "Run bookmarklet?" the first time — just confirm

Drag the button below to your bookmarks bar (View → Show Bookmarks Bar, or Cmd Shift B if hidden).

bookmark_add Save to Golem OS

Drag this button to your bookmarks bar

💡

Can't drag? Right-click the button → "Bookmark Link" — it'll appear in your bookmarks and you can move it to the bar.

How to use it

1

Open any LinkedIn job posting

Works on linkedin.com/jobs/view/… pages

2

Click "Save to Golem OS" in your favorites

A small window opens with job title, company, and URL pre-filled

3

Set a status, add notes, hit Save

The window closes automatically — you're back on LinkedIn
